1. Botão Básico Syntactic sugar

O botão mais simples. Suporta click, dblclick, focus, blur nativamente.

Click me
<!-- HTML --> <lumina-button variant="glass" intensity="intense" accent-color="#7c5cff"> Click me </lumina-button> <script type="module"> const btn = document.querySelector('lumina-button'); // Click event (LuminaUI custom) btn.addEventListener('lumina-press', (e) => { console.log('Pressed!', e); }); // Native double-click btn.addEventListener('dblclick', () => { console.log('Double-clicked!'); }); // Keyboard: Enter or Space triggers click btn.addEventListener('keydown', (e) => { if (e.key === 'Enter' || e.key === ' ') { console.log('Keyboard activation:', e.key); } }); </script>
import 'lumina-ui'; export function BasicButton() { return ( <lumina-button variant="glass" intensity="intense" accent-color="#7c5cff" onLuminaPress={(e) => console.log('Pressed!', e)} onDblClick={() => console.log('Double-clicked!')} > Click me </lumina-button> ); }

11. Toggle Button On/Off

Botão que alterna entre pressed e released. Glow muda com o estado.

Modo escuro
<lumina-toggle-button variant="glass" intensity="intense" accent-color="#ffd166" > Modo escuro </lumina-toggle-button> <script type="module"> const tb = document.querySelector('lumina-toggle-button'); tb.addEventListener('lumina-change', (e) => { console.log('Toggle:', e.detail.pressed ? 'ON' : 'OFF'); if (e.detail.pressed) { document.body.style.background = '#000'; } else { document.body.style.background = '#fff'; } }); // Ler/definir estado via JS console.log('Pressed:', tb.hasAttribute('pressed')); tb.setAttribute('pressed', ''); // Forçar ON </script>

16. Gesture Button Gestos

Reconhece: hold (segure 0.5s), swipe (arraste), double-tap (2x rápido).

Toque • segure • arraste • 2x
<lumina-gesture-button variant="neural" accent-color="#ff6ec7" gestures="hold,swipe,double-tap" > Toque • segure • arraste </lumina-gesture-button> <script type="module"> const gb = document.querySelector('lumina-gesture-button'); // Detecta tipo de gesto gb.addEventListener('lumina-gesture', (e) => { console.log('Gesto:', e.detail.type); // e.detail.type = "hold" | "swipe" | "double-tap" if (e.detail.type === 'hold') { console.log('Menu de contexto...'); } else if (e.detail.type === 'swipe') { console.log('Próximo item...'); } else if (e.detail.type === 'double-tap') { console.log('Favoritar!'); } }); </script>
